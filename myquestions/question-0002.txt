PREGUNTA 2
El Problema:

Recibes un DataFrame con múltiples variables predictoras, pero no todas son relevantes para el modelo. Incluir variables irrelevantes puede afectar el rendimiento del algoritmo.

Debes implementar un proceso que seleccione automáticamente las variables más importantes y entrene un modelo de clasificación.

Tu Misión: Escribe una función llamada pipeline_seleccion_modelo(X, y, k) que realice lo siguiente:
Escale las variables numéricas.
Seleccione las k mejores variables utilizando un método estadístico.
Entrene un modelo de clasificación basado en árboles.
Identifique cuáles variables fueron seleccionadas.
Obtenga la importancia de cada variable en el modelo.
La función debe devolver:
Un diccionario con:
Lista de variables seleccionadas
Importancia de cada variable
